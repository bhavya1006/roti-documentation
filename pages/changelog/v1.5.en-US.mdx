# Changelog

Date: June 11, 2022

<Steps>

### Voicemove Command

You now have the ability to move users between voice channels using the new `/voicemove` command.

### Added R.O.T.I Support

Text-in-voice channels now support R.O.T.I functionality.

### Button Roles

Button roles now support link buttons, providing users with additional interactive buttons.

### Ticket Category Customization

Users can now specify different ticket categories when claimed, allowing for more precise organization.

### Other Changes

Various miscellaneous improvements and bug fixes have been implemented to enhance overall performance and stability.

</Steps>
